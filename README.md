# Show do Milhão

Show do Milhão é um game show brasileiro de perguntas e respostas, que concede um prêmio máximo de um milhão de reais.

## Funcionalidades

- Cadastro/Login
- Tela de Game
- Tela de Perfil
- Envio de Perguntas
